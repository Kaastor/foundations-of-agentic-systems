# Zestaw startowych twierdzeń

Użyj tego zestawu, jeśli chcesz mieć stały punkt wyjścia do Zadania 4.

Możesz też wygenerować własny zestaw twierdzeń przy pomocy chatbota, ale ta wersja startowa jest dobrym domyślnym wyborem dla początkujących.

## Zestaw twierdzeń do audytu

1. Capstone agent wyraźnie dowodzi, że refleksja automatycznie poprawia jakość przeglądu.
2. Końcowy przegląd jest dobrze ugruntowany, ponieważ system użył kilku narzędzi.
3. Przebieg ma wystarczająco dużo materiału, aby uzasadnić twierdzenia o pamięci i planowaniu.
4. System zweryfikował, że każde główne twierdzenie w końcowym przeglądzie miało wsparcie w cytowaniach.
5. Przebieg pokazuje, że samo użycie narzędzi wystarcza, aby odpowiedź była godna zaufania.
6. Odpowiedź końcowa jest ograniczona, ponieważ krok weryfikacji przeszedł pomyślnie.

## Jak używać tego zestawu

Dla każdego twierdzenia zdecyduj, czy jest:

- ugruntowane,
- słabo podparte,
- niepodparte.

Następnie popraw całą odpowiedź do wersji bardziej ograniczonej i lepiej podpartej materiałem źródłowym.

## Po co ten zestaw istnieje

Nie chodzi o to, żeby Cię złapać.
Chodzi o to, żeby dać jeden konkretny punkt startowy, dzięki któremu możesz ćwiczyć samą umiejętność audytu:

- porównywać twierdzenia,
- sprawdzać artefakty,
- oddzielać materiał źródłowy od pewności siebie,
- tworzyć węższy końcowy osąd.
