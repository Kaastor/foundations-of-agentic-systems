## 1. Task

## 2. Candidate Options Or Initial Answer

Wypisz:

- swój pierwszy typ dotyczący najwcześniejszego rozstrzygającego punktu dowodowego,
- co najmniej jeden wcześniejszy, sensowny, ale słabszy punkt kandydujący.

## 3. Comparison Criteria

Powiedz, jak będziesz oceniać „najwcześniejszy rozstrzygający dowód”.
Uwzględnij:

- znaczenie dla granicy zadania,
- widoczność w śladzie albo stanie,
- związek z logiką zatrzymania lub przekazania.

## 4. Evidence

Przywołaj:

- jeden zakres zdarzeń albo krok w `trace`,
- jedną odpowiadającą migawkę stanu,
- jeden artefakt zatrzymania, przekazania albo weryfikacji.

Wyjaśnij, dlaczego słabszy punkt kandydujący nie jest jeszcze rozstrzygający.
