# Przykładowe artefakty

Ten katalog zawiera zapisane referencyjne wyniki wygenerowane przez dostarczone CLI na dostarczonych fixture’ach.

- `spec_review/` — artefakty specyfikacji zadania z `m2a spec-review`
- `run_review/` — przebiegi pojedynczego wariantu z `m2a run-review`
- `compare_architectures/` — porównania wielu wariantów z `m2a compare-architectures`

Te pliki najlepiej czytać razem z `docs/trace-reading.md` i dokumentami slice’ów.
