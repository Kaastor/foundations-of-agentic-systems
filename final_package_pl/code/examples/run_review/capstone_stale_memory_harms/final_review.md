# Przegląd literatury: stale_memory_harms

Wariant: `capstone_agent`

## Zakres

Użyj wcześniejszych notatek, jeśli pomagają, ale zweryfikuj je wobec lokalnego korpusu, aby ocenić, czy refleksja automatycznie poprawia poprawność. Cytuj lokalne dowody i powiedz jasno, jeśli któraś wcześniejsza notatka była nieaktualna.

## Ustalenia według tematów

### Pamięć

Pamięć musi obejmować politykę odczytu i zapominania, a nie tylko przechowywanie. Systemy bogate w pamięć, ale ubogie w narzędzia, utrzymują ciągłość, lecz mają trudność wtedy, gdy potrzebują świeżych dowodów spoza zapisanych notatek. [P04-MemoryPolicy] [P11-MemoryRichToolPoor]

### Refleksja

Refleksja pomaga tylko wtedy, gdy jest powiązana z weryfikacją opartą na dowodach i realną zmianą zachowania. Niekontrolowana stara pamięć sprawia, że refleksja wygląda na bardziej wiarygodną, niż pozwalają na to dowody. [P07-ReflectCheck] [P13-StaleReflection]
