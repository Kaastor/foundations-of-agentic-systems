For setup and reruns, use [STUDENT_SETUP.md](/home/przemek/Biznes/Kursy/course-generation/understanding/runs/Agentic-GenAI-Systems/final_package/STUDENT_SETUP.md).

## Assignment Model

The course uses an AI-native report model:

- chatbot use is normal,
- coding-agent use is encouraged but not required,
- submissions are report-only,
- grading is based on evidence, comparison, critique, bounded revision, and justification.

## What Students Submit

For each assignment, submit one file:

- `submission.md`

Use the assignment-specific `report_template.md`.

Every assignment includes one small `Implementation Lens`.
Assignments 5-6 also require one bounded code change and a `before/after` comparison.
