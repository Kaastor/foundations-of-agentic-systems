# AI Use Guidance

## Chatbots

Useful for:

- proposing candidate earliest evidence points
- helping summarize long trace segments

## Coding Agents

Optional only.
Useful for:

- correlating trace steps with state snapshots
- helping search the artifacts quickly

## Weak AI Use

- asking for a summary of the run and treating that as localization

## Strong AI Use

- asking for multiple plausible earliest evidence points
- checking each against the actual trace, state, and stop artifacts
- defending one point over the others
