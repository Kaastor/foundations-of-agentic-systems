# AI Use Guidance

## Chatbots

Useful for:

- generating two different first-pass task specs
- testing whether an AI overfills ambiguity with guesses

## Coding Agents

Optional only.
Useful for:

- quickly comparing JSON structures
- rerunning `m2a spec-review` if you want stronger evidence

## Weak AI Use

- accepting a polished task spec that silently guesses missing information

## Strong AI Use

- generating multiple candidates
- checking each against the committed course task specs
- revising one bounded final version with explicit ambiguity handling
