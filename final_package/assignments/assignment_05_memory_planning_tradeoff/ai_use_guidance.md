# AI Use Guidance

## Chatbots

Useful for:

- predicting what the stale-memory intervention might do
- helping you articulate comparison criteria before rerunning

## Coding Agents

Optional only.
Useful for:

- locating the exact memory-policy line to edit
- diffing before and after run artifacts quickly

## Weak AI Use

- asking AI to propose a patch and then trusting it without checking the rerun artifacts

## Strong AI Use

- generate a prediction before the change
- use AI to help locate the bounded intervention
- verify the actual effect from before/after artifacts
- be willing to conclude that the intervention harmed the system
