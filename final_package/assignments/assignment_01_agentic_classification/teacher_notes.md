# Teacher Notes

## Why It Is Here

This assignment converts Meeting 1 from vocabulary exposure into artifact-backed classification.

## What A Strong Report Should Contain

- an explicit initial answer plus at least one alternative
- real use of the comparison matrix
- trace-based support for why one variant crosses the threshold
- a short implementation lens showing how baseline flow differs from the control-loop path
- a bounded statement of what is not enough by itself, such as tool use alone

## Common Weak Patterns

- saying “capstone_agent is agentic because it is advanced”
- calling any tool use agentic
- relying only on the prose docs and ignoring artifacts

## Grading Nuance

The exact threshold wording may differ across good answers.
What matters is whether the threshold is defended from the course’s architecture distinctions.
The implementation lens does not need deep code commentary; one accurate structural connection is enough.
