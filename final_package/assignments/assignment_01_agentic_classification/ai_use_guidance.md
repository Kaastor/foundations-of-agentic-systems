# AI Use Guidance

## Chatbots

Useful for:

- producing a first-pass classification
- generating alternative definitions of “agentic”
- forcing yourself to compare two plausible criteria sets

## Coding Agents

Optional only.
Useful for:

- helping you inspect multiple trace files quickly
- helping you summarize where variants differ

## Weak AI Use

- asking for a definition and copying it
- using generic agent terminology without checking the course artifacts

## Strong AI Use

- asking for multiple candidate classifications
- checking those candidates against the actual comparison matrix and traces
- revising after noticing what the initial AI answer ignored
