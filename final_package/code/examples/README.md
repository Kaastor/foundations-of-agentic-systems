# Example artifacts

This directory contains committed reference outputs produced by the shipped CLI over shipped fixtures.

- `spec_review/` — task-spec artifacts from `m2a spec-review`
- `run_review/` — single-variant runs from `m2a run-review`
- `compare_architectures/` — multi-variant comparisons from `m2a compare-architectures`

These files are meant to be read alongside `docs/trace-reading.md` and the slice docs.
