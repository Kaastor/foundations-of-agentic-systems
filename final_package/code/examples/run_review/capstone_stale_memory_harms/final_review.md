## Findings by topic

### Memory
Memory must include retrieval and forgetting policy, not just storage. Memory-rich tool-poor systems preserve continuity but struggle when they need fresh evidence outside stored notes. [P04-MemoryPolicy] [P11-MemoryRichToolPoor]

### Reflection
Reflection helps only when tied to evidence-backed verification and changed behavior. Unchecked stale memory makes reflection look more reliable than the evidence supports. [P07-ReflectCheck] [P13-StaleReflection]
